#### 增加纬度和减少纬度

1. 增加纬度

```python
a = tf.random.normal([4, 35, 8])
tf.expand_dims(a, axis=0).shape    # 要增加一个纬度，必须先增加一个轴，这行就是在第一纬度添加一列
```

2. 减少纬度

   ```pythonø
   a = tf.zeros([1, 2, 1, 3])
   
   tf.squeeze(a, axis=0).shape    # 将纬度为0的那一列删掉
   # TensorShape([2, 1, 3])
   ```
